* **OS:** macOS btw
* **Text Editor:** Neovim
* **IDE:** IntelliJ 
* **System Monitor:** Bottom
* **Colorscheme:** Nord or [darkside](https://github.com/shahmilav/darkside-iTerm2)
* **Font:** Iosevka SS02 (Anonymous Pro Style)
